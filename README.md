В итоговом проекте 6 спранта мною реализовано UI-тестирование сервиса «Яндекс.Самокат», с применением Page Object Model.
Структура проекта:

allure_results - Allure-отчёт
helper - хранит вспомогательные функции для генерации данных создания заказа
locators - содержит файлы с локаторами
pages - содержит методы и шаги
tests - содержит тестовые сценарии
curl.py - содержит URL-адреса используемых страниц
data.py - содержит тестовые данные

В свою очередь папка tests содержит в себе:
conftest.py - фикстуры

test_important_questions.py 
- тесты, которые проверяют ответы в разделе "Вопросы о важном",
- тест перехода по логотипу "Самоката" на главную страницу "Самоката",
- тест по логотипу Яндекса, открытие в новом окне главной страницы Дзена.

test_scooter_order.ру
-тест заказа самоката с помощью кнопки "Заказать" сверху страницы
-тест заказа самоката с помощью кнопки "Заказать" внизу страницы
